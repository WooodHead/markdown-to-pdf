# Convert markdow to pdf ## Use the online editor * write your markdown directly here * or drag and drop a markdown file directly here then hit the "Convert to PDF!" button! We use a beautiful stylesheet, so your markdown text will look amazing! ## Use the API Or you can convert from the command line using our API: example with curl: ```bash # convert doc.md to doc.pdf curl --data-urlencode "data=`cat doc.md`" "https://markdown-to-pdf.herokuapp.com" > doc.pdf ```
